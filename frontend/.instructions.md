# Frontend — Copilot Instructions

You are working in the **frontend** of Zault, a React + TypeScript application built with Vite.

## Key Context

- Read `CONTEXT.md` (root) for system architecture.
- Read `frontend/CONTEXT.md` for frontend-specific decisions and state.
- **Read `docs/api/openapi.yaml` for all API endpoints, request/response shapes.**
- Generated TypeScript types are in `frontend/src/api/schema.d.ts`.

## Do NOT

- **Do NOT read backend Java code.** Use `docs/api/openapi.yaml` as the single source of truth for the API.
- Do NOT suggest yarn or pnpm — this project uses npm.

## Conventions

- Functional React components, TypeScript strict mode
- API base URL: `/api/` (proxied by nginx in production, Vite proxy in dev)
- CLI commands: use `make` targets (see root `Makefile`)
- Regenerate API types: `make api-types`

## API Types

Types are auto-generated from the OpenAPI spec:
```bash
make api-types  # generates frontend/src/api/schema.d.ts
```
Import and use these types instead of defining API shapes manually.
